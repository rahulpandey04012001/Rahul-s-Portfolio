Projects

I have worked on diverse projects including a tech blogging platform (ThirstForTech), an AI Voice Assistant named June with features like Google search and messaging, and a fintech B2B Invoice Management System built during my internship at HighRadius. I also contributed to a finance domain automation framework at Coforge for SEI, focusing on stability, accuracy, and client-centric delivery. These projects reflect my full stack, automation, and AI integration skills, with more advanced projects planned ahead.

AI-powered Test Case Generator

During a 24-hour hackathon, I developed an AI-powered Test Case Generator that automates test case creation based on user stories and acceptance criteria. The tool uses Natural Language Processing to extract key scenarios from functional requirements and generates structured test cases, reducing manual effort and increasing coverage. Built using JavaScript, Node.js, and integrated with a simple UI, the solution was well-received for its real-world applicability in QA and Agile environments.

AI enabled B2B Invoice Management System

As part of my internship at HighRadius, I developed an AI-enabled B2B Invoice Management System designed to streamline the invoice processing cycle for enterprises. The system uses machine learning algorithms to predict payment dates, manage customer invoices, and prioritize collections. Built using Java, Spring Boot, Hibernate, MySQL, and REST APIs, the project handles thousands of records efficiently and enhances the accuracy and speed of financial workflows.
